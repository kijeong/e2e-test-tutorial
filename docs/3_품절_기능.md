## 작업 프롬프트: “품절(Out of stock) 기능 + UI/구매 제어”

### 목표

쇼핑몰 앱에 **품절 상태**를 추가하고, UI/장바구니/결제 흐름에서 품절 상품을 올바르게 처리한다.

### 요구사항(기능)

- **상품 데이터**
  - 품절 상태를 표현할 수 있어야 한다.
  - 기준은 `db.json`의 `products.isOutOfStock`을 사용(예: `isOutOfStock === true` 이면 품절).

- **상품 목록/상세 UI**
  - 품절인 상품은 목록/상세에서 **“품절” 표시**가 보인다.
  - 품절인 상품은 **장바구니에 담을 수 없다**.
    - “장바구니 담기” 버튼을 disabled 처리하거나, 클릭 시 안내 메시지(토스트 등)를 노출한다.

- **장바구니/결제 UI**
  - 장바구니에 담긴 상품 중 하나라도 품절이면 **“구매하기(checkout)” 버튼이 비활성화**되어 결제를 진행할 수 없다.
  - (선택) 장바구니에서 해당 품절 상품이 명확히 보이도록 표시한다.

### 구현 범위(수정 후보 파일)

- 데이터: `db.json`
- 상품 렌더링: `src/components/ProductCard.tsx`, `src/pages/ProductDetailPage.tsx`(존재 시)
- 장바구니/결제 버튼 로직: `src/pages/CartPage.tsx`, `src/store/CartContext.tsx`

### 완료 조건(Definition of Done)

- 품절 표시가 UI에서 확인된다.
- 품절 상품은 장바구니 담기 불가.
- 장바구니에 품절 상품이 포함되면 구매 불가(구매 버튼 비활성화).
